1. 镜像加速

vim /etc/docker/daemon.json
{
  "registry-mirrors": [
    "https://hub-mirror.c.163.com",
    "https://mirror.baidubce.com",
    "https://registry.docker-cn.com",
    "https://reg-mirror.qiniu.com",
    "https://dockerhub.azk8s.cn",
    "https://docker.mirrors.ustc.edu.cn"
  ]
}
重启docker
sudo systemctl daemon-reload
sudo systemctl restart docker
docker info # 查看有没有修改成功

2. 镜像下载

先从国内镜像站pull，比如 `docker pull mysql:5.7.41 `

docker pull hub-mirror.c.163.com/library/mysql:5.7.41
docker tag hub-mirror.c.163.com/library/mysql:5.7.41 mysql:5.7.41

3. 镜像导出

docker save -o mysql-5.7.41.image mysql:5.7.41

4. 镜像导入

docker load -i mysql-5.7.41.image
