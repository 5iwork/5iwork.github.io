文档格式转换

本页目录
  • 1. Word → Asciidoc
  • 2. Markdown → Asciidoc
  • 3. asciidoc → word

文档格式转换之王，想见Pandoc官网。

1. Word → Asciidoc

pandoc --from=docx --to=asciidoc --wrap=none --atx-headers --extract-media=extracted-media 文件名.docx > 文件名.adoc

2. Markdown → Asciidoc

pandoc -f markdown -t asciidoc -o index.adoc index.md

3. asciidoc → word

# 使用asciidoctor转换adoc为docbook格式
asciidoctor -a sectnums -a icons=font -a toclevels=6 -b docbook index.adoc -o index.xml
# 使用pandoc转换为docx格式
pandoc -f docbook -t docx -o index.docx index.xml
