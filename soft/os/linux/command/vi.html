4. 大小写转换

gu

小写,前后都可以指定范围

gU

大写

0gu$

当前行全部转小写：可以分三部分解释 [0]：光标定位到行首 gu：选定范围统统变小写 [G]一直操作到行尾

1GguG

整个文档全部转小写：可以分三部分解释 [1G]：光标定位到文件首 gu：选定范围统统变小写 [G]一直操作到文件结束部分
