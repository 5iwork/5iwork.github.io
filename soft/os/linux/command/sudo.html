1. sudo的环境变量

在linux系统中，当你使用sudo去执行一个程序时，出于安全的考虑，这个程序将在新的、最小化的环境中执行。这会导致我们经常碰到 sudo 执行命令时报 command not found。

/etc/sudoers
Defaults    env_reset (1)
Defaults    env_keep += "JAVA_HOME" (2)
Defaults    secure_path = /sbin:/bin:/usr/sbin:/usr/bin (3)
1 重置环境变量，如果不想重置可以使用 Defaults !env_reset
2 需要保留的环境变量
3 默认$PATH变量
