安装包会自动创建各自的用户，并将程序目录和数据目录都设置为该用户所有，所以建议用root用户执行安装程序，默认目录一般为/opt目录和/usr/lib目录，建议自定义设置到统一一个公共权限的目录，比如/home目录下，这样不会有权限问题。目录规划如下：

/home目录
├── crowd
│   ├── application-data
│   └── application
├── confluence
│   ├── application-data
│   └── application
└── jira
    ├── application-data
    └── application

  1. 访问地址：https://gitee.com/pengzhile/atlassian-agent

  2. 解压并配置

mkdir /var/lib/atlassian && tar -xzvf atlassian-agent-v1.3.1.tar.gz -C /var/lib/atlassian
echo 'export JAVA_OPTS="-javaagent:/var/lib/atlassian/atlassian-agent-v1.3.1/atlassian-agent.jar ${JAVA_OPTS}"' > /etc/profile.d/atlassian-agen.sh

  1. 以上配置当用 service xxx start 启动时会不生效，建议直接修改tomcat的启动脚本
    .<安装目录>/bin/catalina.sh

export CATALINA_OPTS="-javaagent:/var/lib/atlassian/atlassian-agent-v1.3.1/atlassian-agent.jar $CATALINA_OPTS"
