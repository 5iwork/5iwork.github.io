1. 概述

JSON作为目前最常用的前后端数据交互格式，具有简洁、可读等优势，我们日常使用时也大部分不会对JSON数据做约束，但有些时候，我们需要对JSON数据做校验或者生成标准代码时，就需要能精确定义JSON的数据结构和约束，和 XML Schema 一样，Json Schema 就是用来定义 json 数据约束的一个标准。
