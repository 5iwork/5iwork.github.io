1. 获取远程特定分支

//查看远程分支
git branch -r
//创建本地分支并关联
git checkout -b 本地分支 origin/远程分支

//已有本地分支创建关联
git branch --set-upstream-to origin/远程分支名  本地分支名
//拉取
git pull

2. 两个不同历史的仓库或分支合并

git merge xxx --allow-unrelated-histories
